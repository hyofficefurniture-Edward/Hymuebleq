# Website Strategy

- Last updated: 2026-06-29
- Main goal: local preview of a high-quality Spanish B2B independent site

## Website Goal

The site must help a Spanish-speaking B2B buyer quickly understand:

1. What Hymueble supplies.
2. Which project sectors it supports.
3. Why a China-based factory supply path can be credible.
4. How to inspect solutions through a Virtual Showroom.
5. How to submit project requirements for quotation.

## Conversion Paths

- Primary: Home or sector page -> Contact form -> success feedback.
- Secondary: Home or catalog page -> Catalog lead modal -> success feedback.
- Showroom path: Home / Hoteles / Catalogo hoteleria -> Virtual Showroom link -> Contact.
- Trust path: Home -> Factory -> Process / Quality Control -> Contact.
- Evidence path: Home -> Projects -> Project detail -> Contact.

## B2B Decision Logic

B2B buyers are evaluating risk before price. The site should answer:

- Can this supplier handle my sector?
- Can they customize by project?
- Do they understand manufacturing and export?
- Can I see real spaces and showroom references?
- What information should I submit to get a useful quote?

## Page Priority

- S: `/`, `/hoteles/`, `/contacto/`
- A: `/catalogo/hoteleria/`, `/proyectos/hoteleria/`, `/fabrica/`, Virtual Showroom entry
- B: `/oficinas/`, `/salud/`, `/educacion/`, `/residencial/`, sector catalog/project pages
- C: `/catalogo/`, `/proyectos/`, `/recursos/`, `/recursos/blog/`, `/recursos/videos/`
- D: hotel category pages and project details, currently light but clickable

## CTA Strategy

- Use `Solicitar cotización` for project intent.
- Use `Enviar requerimientos del proyecto` for detailed B2B inquiries.
- Use `Explorar showroom virtual` for high-intent visual exploration.
- Use `Solicitar catálogo` for lead capture.
- Use `Hablar con un asesor` only when real contact details are ready.

## Non-Goals For Current Preview

- No real backend.
- No CMS.
- No production deployment.
- No real personal contact details.
- No unverified certificate claims on the live frontend.
