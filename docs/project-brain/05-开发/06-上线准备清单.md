## 推荐上线节奏

### 阶段 1：老板确认

- 使用本地预览演示当前 v0.93。
- 确认品牌名、视觉方向、主行业优先级和是否先部署测试域名。
- 收集真实 WhatsApp、邮箱、社媒账号；如需品牌手册级资产，再补 SVG/AI logo。

### 阶段 2：可外发测试版

- 替换真实联系方式。
- 接入表单接收。
- 使用 `.env.local` 或部署平台环境变量配置 `PUBLIC_HYMUEBLE_WHATSAPP_URL`、`PUBLIC_HYMUEBLE_EMAIL`。
- 部署到测试域名或临时正式域名。
- 做桌面/移动端人工验收。

### 阶段 3：正式上线版

- 补图册 PDF、案例字段、证书资料和隐私政策。
- 更新 sitemap/canonical 到正式域名。
- 接入统计和转化事件。
- 做最终上线前验收。

## 不建议现在做

- 不为了上线速度把未确认案例数据写成确定事实。
- 不把其他站群入口重新放进 Footer。
- 不在没有真实接收方案时假装表单已发给销售。
- 不把未经压缩的大 PDF 直接挂到页面，避免加载和留资路径失控。
