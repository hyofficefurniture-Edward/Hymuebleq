## 0. 本轮边界

- 本轮只做“拉美/西语同行表达 + B2B 信任心理学”研究包，并选 `/contacto/` 做一页试改稿。
- 不改源码、不改 URL、不改页面结构、不改视觉设计、不动 SEO 主关键词。
- 同行网站只用于吸收表达模式和业务语境，不复制原文。
- 以下数据已由业务确认，本轮不再标记为不确定：24h 响应 SLA、表单真实接收端、图册 PDF 发送流程、10.000/40.000 m2 展厅口径、工厂面积/员工/项目数、证书适用范围、付款比例、MOQ、5-8 周交期、样品收费规则、BBK 35 天交付、RMB/面积/床位/房间数公开口径、安装/售后/物流责任边界。

## 1. 结论先行

当前 `/contacto/` 的方向已经比典型 AI 文案更接近真实 B2B 采购沟通：它有 `sector / ciudad / WhatsApp / cantidades / planos / lista FF&E / catálogo / showroom`，也说明“细节可以后补”。问题不在于大面积重写，而在于局部语气仍偏“模板说明”，可以更像真人销售/项目经理在引导客户发起询价。

SEO 优先级需要前置：`cotización / muebles / proyecto` 这类核心词应尽量出现在 H1、首屏第一句、段落开头和 CTA 前半句。去 AI 化只能调整语气和信息密度，不能把主关键词移到句尾或改成过于口语的表达。

建议本页采用“慢改”：保留现有结构和关键词，只微调 6-8 处句子。目标不是变得更华丽，而是让客户更明确：

- 我现在只要给哪些基本信息；
- 没有完整清单也可以先开始；
- 图纸、FF&E、数量和参考图什么时候有用；
- 发送后 Hymueble 会怎样跟进；
- 这不是零售购买，而是项目型 `cotización`。

## 3. 可吸收的本地表达规则

### 3.1 CTA 语气

拉美 B2B 家具网站常见 CTA 不是单一 `Contáctenos`，而是更接近采购动作：

- `Cuéntanos tu proyecto`
- `Cotiza tu proyecto`
- `Solicita una cotización`
- `Solicitar una propuesta`
- `Hablar por WhatsApp`
- `Solicitar catálogo`

对 Hymueble 的启发：CTA 要保留 `cotización / proyecto / catálogo / WhatsApp / muebles`，不能改成空泛品牌沟通。

### 3.2 动词选择

同行更常用可执行动词，而不是只用形容词：

- `fabricamos`
- `diseñamos`
- `equipamos`
- `cotizamos`
- `revisamos`
- `coordinamos`
- `preparamos`

对 Hymueble 的启发：工厂型供应商要多说“我们做什么、客户给什么、下一步是什么”，少说 `excelente / integral / profesional / confiable`。

### 3.3 采购材料表达

更自然的项目采购信息包括：

- `planos`
- `medidas`
- `lista FF&E`
- `cantidades aproximadas`
- `fotos de referencia`
- `acabados`
- `materiales`
- `fecha de apertura`
- `espacios a equipar`

对 Hymueble 的启发：这些词比“高品质解决方案”更能留住真实采购客户。

### 3.4 反例规则

本轮应避免：

- `soluciones integrales` 单独出现，但没有项目流程、材料、质检、物流或安装边界支撑；
- `líder en la industria`、`comprometidos con la excelencia`、`calidad superior` 这类模板品牌话术；
- `compra en línea`、购物车、SKU、折扣、库存、单品价格等零售电商口吻；
- 过度 `premium / lujo / exclusivo`，除非页面有明确高端酒店或设计证据；
- 把中国工厂出口站写成本地装修设计公司。

### 3.5 Google SEO 关键词前置规则

本轮所有试改稿增加一条硬规则：关键词不仅要保留，还要尽量靠前。

- H1 优先保留 `cotización de muebles`、`muebles para hoteles`、`muebles de oficina`、`muebles a medida` 等主搜索词。
- 首屏第一句话前 8-12 个词内尽量出现主关键词或强相关采购词。
- H2 可以更自然，但不能把产品类别、项目场景、工厂能力、定制能力和询价路径弱化。
- CTA 前半句优先出现 `cotización / proyecto / catálogo / WhatsApp`，避免只写 `Cuéntanos`、`Hablemos`、`Conversemos`。
- 若“自然表达”和“关键词前置”冲突，联系页、品类页和首页首屏优先 SEO；说明段落和帮助文案再承担自然化。

### 3.6 三重验收规则

后续每一句西语试改都必须同时通过三关：

1. SEO：主关键词或强相关采购词尽量靠前，不能为了自然化删除 `cotización / muebles / proyecto / catálogo / WhatsApp`。
2. 拉美 B2B 语感：优先使用 `cotizar / cotización / proyecto / catálogo / WhatsApp / planos / cantidades / referencias / lista FF&E`，少用西班牙本土或过度正式的机构腔。
3. 低心理压力：尽量使用 `puedes enviar / si ya tienes / no son obligatorios / revisamos después / datos iniciales`，避免让客户感觉“一次必须准备完整资料才配询价”。

## 5. `/contacto/` 当前文案总体判断

### 5.1 适合保留的部分

- 页面主 SEO 方向明确：`Solicitar cotización de muebles para proyectos`。
- H1 和 meta description 保留了 `cotización / muebles / proyectos`。
- 表单不是零售电商口吻，没有价格、购物车、库存、SKU。
- “先给 5 个信息，技术细节后补”的策略符合低摩擦询盘。
- `planos / lista FF&E / cantidades / fotos de referencia` 已经出现，是真实采购语言。
- WhatsApp、catalogo、showroom 入口都合理。

### 5.2 主要问题

- 局部句子仍像“网站说明文字”，不是项目销售在引导客户。
- `Completa una consulta rápida` 稍微像模板表单话术，可以改成更自然但仍关键词前置的表达，例如 `Para cotizar muebles...`，不建议只改成 `Comparte...`。
- “los detalles pueden venir después”语义正确，但可以更口语化、少翻译感。
- 目录弹窗标题 `Descarga el catálogo de mobiliario` 可再贴近项目：客户不只是下载，而是希望拿到适合行业的图册。
- 共享 CTA 的 `cotización gratuita` 可留待下一批讨论；对 B2B 项目而言，`gratuita` 可能有轻微零售/获客广告感，但不必本轮动。

## 9. 不建议本轮改的内容

- 不动 `src/components/CTA.astro`，因为它是共享组件，会影响全站。
- 不动付款比例、MOQ、交期、样品、证书、项目金额、面积、安装/售后/物流边界。
- 不把 `/contacto/` 改成“销售落地页”；它应该保持询价入口，不做大段说服。
- 不引入 `líder / excelencia / calidad superior / soluciones integrales`。
- 不引入客户名称、国家数量、产能数字、证书范围以外的事实。
