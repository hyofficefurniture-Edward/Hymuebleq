## 第一阶段执行口径（v0.1 对齐）

- S 类：保留现有页面结构但必须按策划案重审文案/模块（首页、酒店页、表单页）。
- A 类：继续落地并补齐策划要求细化项（酒店图册、项目页、工厂页）。
- B 类：先做轻量版入口与路由通路，暂不追求全部板块上线。
- C 类：补齐枢纽和资源入口，承担内链、SEO 结构和演示时的点击闭环。
- D 类：不做最终长详情，但必须有真实图片、页面标题、项目/目录说明、CTA 和返回路径。

## 当前范围判断

- 品牌展示统一为 `Hymueble`；策划案中 `Hymuebles` 视为旧写法或笔误，暂不在页面前台使用。
