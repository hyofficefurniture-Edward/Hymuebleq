# Hymueble 页面审查清单 v1

来源：`鸿业家具西班牙语独立站策划案.xlsx`

用途：先用于页面内容与排版审查，不直接代表最终上线版本。每个页面确认后，再进入开发修改。

## 建议审查顺序

1. `/` 首页
2. `/hoteles/` 酒店汇总页
3. `/catalogo/hoteleria/` 酒店图册页
4. `/proyectos/hoteleria/` 酒店项目页
5. `/contacto/` 联系页
6. `/fabrica/` 工厂页
7. `/catalogo/` 图册汇总页
8. `/proyectos/` 项目汇总页
9. `/recursos/`、`/recursos/blog/`、`/recursos/videos/` 资源页体系
10. `/oficinas/`、`/salud/`、`/educacion/`、`/residencial/` 第二梯队行业页

## 全局导航与页脚

### 导航

排版样式：
- 顶部悬浮导航。
- 左侧品牌 Logo。
- 中间主导航：Inicio、Hoteles、Oficinas、Salud、Educación、Residencial、Fábrica、Recursos、Contacto。
- 右侧主 CTA：`Solicitar cotización gratuita`，跳转 `/contacto/`。
- 预留语言切换入口。

审查重点：
- 导航是否更偏 B2B 采购，而不是普通官网。
- 主 CTA 是否足够明显。
- 第一阶段酒店路径是否比其他栏目更突出。

### 页脚

排版样式：
- 顶部放一句转化文案：`Cuéntanos los detalles de tu proyecto. Te enviamos una cotización en menos de 24 horas.`
- 放 CTA：`Solicitar cotización gratuita`，跳转 `/contacto/`。
- 下方放公司 Logo、快速导航、社媒入口、联系入口、备案与版权信息。

审查重点：
- 页脚是否要直接放简化联系表单。
- 是否需要 WhatsApp 固定入口。

## `/` 首页

页面目标：
- 让首次进入网站的项目采购、设计公司、建筑公司、经销商、进口商快速判断 Hymueble 是可靠的中国工厂直供供应商。
- 转化路径：Banner 定位 B2B -> 信任数据 -> 五大品类分流 -> 案例验证 -> 工厂和证书背书 -> WhatsApp 或表单询盘。

页面结构与排版：
1. Hero/Banner
   - 大背景图加暗色遮罩，优先用真实酒店项目图、办公空间图或工厂航拍图。
   - H1：`Muebles para proyectos B2B en Latinoamérica, directo de fábrica en China`
   - 副标题强调：中国工厂直供、定制、规模化生产、出口标准、物流支持。
   - bullet 信任点：无中间商、40+ 年制造经验、五大行业、质控认证。
   - CTA：`Cotizar mi proyecto` -> `/contacto/`
2. 视频介绍
   - 插入公司 YouTube 视频。
   - 视频封面叠加两个 CTA：`Consultar ahora`、WhatsApp 图标。
3. 核心信任条
   - 40+ 年经验、350000 m2 生产车间、40000 m2 展厅、10000+ 项目、2000+ 集团人员。
   - 适合做成横向数据条或深色信任带。
4. 五大品类总览
   - H2：`Mobiliario para cada tipo de proyecto`
   - 两大卡：酒店、办公。
   - 三小卡：医疗、教育、住宅。
   - 图片卡片悬浮动画，点击跳转对应行业页。
5. 真实项目案例
   - H2：`Proyectos reales de mobiliario B2B`
   - 展示 4-6 个案例卡片：2 个酒店、2 个办公、1-2 个拉美项目。
   - 卡片字段：名称、国家/地区、行业、主图、`Ver detalles del proyecto`、`Consultar un proyecto similar`。
6. 项目服务流程
   - H2：`De la consulta a la entrega: solución integral para tu proyecto de mobiliario`
   - 流程：报价、现场测量、设计、饰面确认、样板、生产、质检、终检、包装、物流、安装、售后。
   - 适合横向流程图或分组步骤卡。
7. 工厂直销与制造能力
   - H2：`Fabricación directa en China, sin intermediarios`
   - 使用工厂外观、车间、加工、质检包装、设备图做 6 图滚动。
   - 图标卖点：大规模、定制化、强质检、出口包装与文件、五大品类。
   - 双 CTA：`Conocer la fábrica` -> `/fabrica/`，`Consultar fabricación a medida` -> `/contacto/`。
8. 认证与合作
   - 证书/合作品牌 Logo 滚动或网格。
9. FAQ
   - 四组：产品与制造、质量与信任、物流与进口、联系流程。
10. 文章展示
   - 策划案标记为“不急加”，第一阶段可后置。

审查重点：
- 首页是否要把视频提前到 Hero 之后。
- 数据条数字是否全部确认沿用。
- 真实案例名称是否可直接公开。

## `/hoteles/` 酒店汇总页

页面目标：
- 酒店行业客户进入后的主承接页，说明 Hymueble 能做酒店项目家具的一站式供应。

页面结构与排版：
1. Hero/Banner
   - H1：`Mobiliario para Hoteles - Hymuebles`
   - 副标题：`Proveedor de mobiliario hotelero para proyectos B2B directo desde fábrica en China`
   - 正文强调从中国制造到交付到项目现场的全过程管理。
   - CTA：`Contactar ahora` -> `/contacto/`
2. 酒店家具目录模块
   - H2：`Catálogo de Muebles para Hotelería`
   - 网格卡片展示：Habitaciones、Muebles de Baño、Exteriores、Muebles para Bar、Muebles para Restaurante、Mobiliario para Conferencias。
   - 每张卡片：真实空间图 + 空间名称 + 简短描述 + `Ver más`。
   - 点击跳转对应目录子页，例如 `/catalogo/hoteleria/habitaciones/`。
3. 酒店展厅模块
   - H2：`Showroom Virtual de Mobiliario Hotelero`
   - 背景或大图区域嵌入 VR 展厅入口。
   - CTA1：`Ver showroom virtual` -> Matterport 链接。
   - CTA2：`Agendar visita en persona` -> `/contacto/`。
4. 酒店案例模块
   - H2：`Proyectos de Mobiliario Hotelero`
   - 案例卡片字段：项目名、国家城市、交付年份、酒店类型、客房数量、供应家具类型、材质/工艺。
   - CTA1：`Explorar proyectos` -> `/proyectos/hoteleria/`
   - CTA2：`Consultar un proyecto similar` -> `/contacto/`
5. 合作伙伴背书
   - Logo 网格或横向滚动。
6. 服务流程模块
   - 沿用首页流程，但文案聚焦酒店家具项目。

审查重点：
- 酒店汇总页是否作为第一阶段最重要行业页来加厚内容。
- 酒店目录卡片是否只放 6 类，还是增加 lobby/public area。
- VR 展厅放在目录前还是目录后。

## `/catalogo/hoteleria/` 酒店图册页

页面目标：
- 承接想看产品分类和下载图册的采购客户。

页面结构与排版：
1. Hero/Banner
   - H1、副标题、正文策划案留空，需要沿用酒店目录定位补全。
   - CTA：`Cotizar mi proyecto ahora` -> `/contacto/`
2. 酒店家具目录模块
   - 以产品空间分类为主体。
   - 视觉重点是产品/空间卡片，适合 3 列或 2 列响应式卡片。
   - 每张卡片应有：图片、分类名、说明、`Ver más`、`Descargar catálogo`。
3. 酒店展厅模块
   - H2 策划案留空，需要补全。
   - 插入 VR 展厅。
   - CTA1：`Ver showroom virtual`
   - CTA2：`Agendar visita en persona`
4. 图册下载留资弹窗
   - 点击 `Descargar catálogo` 或 `Solicitar catálogo PDF` 打开弹窗。
   - 弹窗表单收集公司/国家城市/WhatsApp/邮箱/行业/需求。

审查重点：
- H1 是否用“Catálogo de muebles para hoteles”这类 SEO 口径。
- 图册下载是否所有卡片都触发留资弹窗。
- 是否需要真实 PDF 下载，还是第一阶段只留资。

## `/proyectos/hoteleria/` 酒店项目页

页面目标：
- 用真实项目证明交付经验，推动客户咨询类似项目。

页面结构与排版：
1. Hero/Banner
   - 策划案中 H1 和正文留空，需要补全。
   - 建议定位：真实酒店家具项目案例。
2. 酒店项目案例陈列模块
   - H2 策划案留空，需要补全。
   - 卡片式陈列，展示项目主图、项目名称、地点、项目类型、供应范围。
   - CTA：`Descubrir el proceso completo` -> 对应真实案例详情页。
3. 合作背书模块
   - 卡片式背书或 Logo 区。

审查重点：
- 第一阶段是否只做案例列表页，不做单个案例详情页。
- 案例名称是否可直接用 Hilton、Kempinski、Huntington 等真实名称。
- 每个案例是否需要“项目证据”：PDF、安装图、视频、官网链接。

## `/contacto/` 联系页

页面目标：
- 完成询盘留资，适合项目采购和经销商填写。

页面结构与排版：
1. 表单主区
   - H1：`¿Tienes un proyecto de mobiliario?`
   - 副标题：`Completa lo esencial — te respondemos en menos de 24 horas.`
   - 表单字段：
     - `Nombre o empresa *`
     - `País o ciudad del proyecto *`
     - `WhatsApp *`
     - `Correo *`
     - `Sector del proyecto *`
     - 开放式项目描述
     - 保密告知
   - 提交按钮：`Enviar mi consulta de proyecto`
   - 成功反馈：`¡Muchas gracias! Recibimos tu consulta y te contactaremos en menos de 24 horas por WhatsApp o correo.`
2. 图片与地图
   - 左侧精品家具图。
   - 右侧地图，链接到工厂地址。

审查重点：
- 是否需要上传附件字段。
- 地图是放 Google Maps 还是用静态地图图像。
- WhatsApp 是否作为单独大按钮固定显示。

## `/fabrica/` 工厂页

页面目标：
- 强化中国工厂直供、生产能力、定制能力、质控和出口能力。

页面结构与排版：
1. Hero/Banner
   - 三图轮播：工厂俯拍、工人特写、家具特写。
   - H1：`Fábrica de mobiliario para proyectos B2B`
   - 副标题：`Más de 40 años fabricando muebles para el mercado global`
   - 正文强调原材料到出口包装的全过程控制。
   - CTA：`Agendar una visita` -> `/contacto/`
2. 公司历史流程图
   - 借鉴办公家具站的工厂历史流程图。
3. 工厂与贸易公司对比
   - H2：`Sin intermediarios: Fabricante directo en China con alcance internacional`
   - 做视觉对比：我们是工厂 -> 客户；别人是多级中间环节。
4. 生产设备能力
   - H2：`Equipos de producción para mobiliario de proyectos`
   - 车间和设备轮播。
   - 分类展示：开料裁板、封边、涂装表面处理、木工定制、金属加工、环保与仓储。
5. 定制化服务模块
   - H2：`Personalización según proyecto`
   - 图纸/产品图纸/服务图。
   - CTA：`Cuéntanos tu proyecto` -> `/contacto/`
6. 证书认证
   - 证书图或认证卡片。

审查重点：
- 工厂页是否第一阶段就做深。
- 设备清单是否全部展示，还是只做能力分类摘要。
- 是否需要突出“不是贸易公司”。

## 第二梯队行业页

范围：
- `/oficinas/`
- `/salud/`
- `/educacion/`
- `/residencial/`
- 对应的 `/catalogo/.../` 和 `/proyectos/.../`

当前策划案状态：
- 工作簿中只明确了导航、汇总页链接和首页五大品类入口。
- 没有像酒店页一样给出完整详细页面结构。

建议第一阶段处理：
- 保留入口页。
- 做“行业定位 + 目录入口 + 案例入口 + 联系 CTA”的轻量页面。
- 等酒店路径确认后，再复制结构扩展到办公、医疗、教育、住宅。

审查重点：
- 是否接受第一阶段做轻量入口页。
- 哪个行业应作为第二个深做页面：办公、医疗、教育、住宅。
