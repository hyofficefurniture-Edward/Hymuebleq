# 修订版上线前关键词动作清单

## 上线前

| 页面 | 动作 | 理由 | 验收 |
| --- | --- | --- | --- |
| /oficinas/ | 将页面关键词中心从 `muebles para oficina` 校准到 `muebles de oficina`，但 Title/H1 用 `Muebles de oficina para proyectos` | Discover 显示 `muebles de oficina` 搜索量更高，但零售意图混入，需要 B2B 限定 | Title/H1/首段不变成零售店语义 |
| /catalogo/oficinas/ | 保持 `catálogo de muebles de oficina` 为主词 | Discover 和历史指标均支持目录词；目录页应与 /oficinas/ 分工 | H1、Title、内链锚文本承接 catálogo |
| /catalogo/oficinas/colaboracion-y-brainstorming/ | 继续把主词改为 `muebles para espacios colaborativos de oficina` | `brainstorming` 有量但意图错配；Discover 支持协作家具词 | brainstorming 只作为场景词，不做 Title/H1 主词 |
| /contacto/ | 保持 `solicitar cotización de muebles` 为主词，加入 `cotización de muebles de oficina` 辅助 | conversion Discover 只有办公报价词返回搜索量，但联系页不能变成办公单页 | 转化页仍服务全站 RFQ |

- 不改 `/catalogo/oficinas/colaboracion-y-brainstorming/` URL，除非上线状态、redirect、sitemap、catalogReferences 都人工确认。
- 不把 `sillas de oficina`、`escritorio`、`sillas de escritorio`、`silla ergonomica` 做行业页主词。
- 不追 `precio`、`barato`、`segunda mano`、品牌词、本地城市词。
- 不把住宅页改成泛家装/豪宅灵感页。
