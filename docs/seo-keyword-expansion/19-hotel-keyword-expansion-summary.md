## Direct answers

1. 为什么上一轮酒店词大多是 0-50？
上一轮种子偏窄，集中在 `proveedor`、`fabricante`、`project/contract`、目录长尾和酒店空间长尾。这些词商业意图强，但自然搜索量通常低。它没有充分覆盖 `sillas para restaurante`、`mesas para restaurante`、`muebles para recepción`、`muebles para exteriores`、`camas para hotel` 这类更宽的产品/空间词。

2. 这是否说明酒店 SEO 没有机会？
不是。机会存在，但不是所有机会都适合做主词。酒店 SEO 的结构应该是：行业页承接 `muebles para hoteles`，目录页承接 catalog/space intent，小类页承接产品和空间词，案例页做 proof。

3. 哪些酒店词有更高搜索潜力？
`muebles para hoteles`、`muebles para hotel`、`camas para hotel`、`muebles para restaurante`、`sillas para restaurante`、`mesas para restaurante`、`sillas para bar`、`muebles para exteriores`、`muebles para terrazas`、`muebles para recepción`、`mobiliario para restaurante`。这些词需要按页面角色加 hotel/project 限定。

4. 哪些酒店词搜索量低但商业价值高？
`proveedor de muebles para hoteles`、`fabricante de muebles para hoteles`、`fábrica de muebles para hoteles`、`mobiliario hotelero a medida`、`muebles para hoteles al por mayor`、`cotización de muebles para hoteles`、`importar muebles para hoteles desde China`。它们适合 H2、正文、FAQ、CTA 和 /fabrica/ 内链。

5. /hoteles/ 应该承接哪些词？
主承接 `muebles para hoteles`。辅助承接 `mobiliario hotelero`、`mobiliario para hoteles`、`muebles para hotel`、`proveedor de muebles para hoteles`、`fabricante de muebles para hoteles`、`mobiliario hotelero a medida`。不要把 `sillas para restaurante` 或 `muebles para exteriores` 做行业页主词。

6. /catalogo/hoteles/ 应该承接哪些词？
主承接 `catálogo de muebles para hoteles`。辅助承接 `catálogo de mobiliario hotelero`、`muebles para hoteles catálogo`、`muebles hoteleros por categoría`、`muebles para lobby de hotel`、`muebles para recepción de hotel`。目录页仍然是参考图/RFQ入口，不是价格库存电商页。

7. 酒店细分类页是否应该接受低搜索长尾词？
应该。酒店细分类页的作用不是每页都追大词，而是把高量泛产品词过滤成项目场景。例如 `sillas para restaurante` 高量但泛，应该落到 `muebles para restaurante de hotel` 页面里做 H2/产品组，而不是单独抢主词。

8. 哪些酒店产品词有流量但可能偏零售？
`sillas para restaurante`、`mesas para restaurante`、`sillas para bar`、`muebles para exteriores`、`sillas de terraza`、`muebles para recepción`、`cabeceras para hotel`、`closets para hotel`。这些词可以做小类页辅助，但必须加 hotel/project/volumen/cotización 语境。

9. 哪些词不适合 Hymueble？
`muebles de hotel usados`、`mobiliario hotel segunda mano`、`muebles para motel`、`mueble para motel`、成人/汽车旅馆相关词、品牌导航词、纯 Pinterest 灵感词、价格/便宜/二手词、家用花园或厨房吧台词。

10. 是否建议马上把酒店关键词应用到页面？
建议，但只做小范围字段应用，不大改正文。酒店组已有足够证据支持 Title/H1/Meta/H2 的校准。

11. 如果建议应用，应该只改哪些字段？
只建议改 Title、H1、Meta Description、H2 方向、内链锚文本、少量首屏关键词前置和 CTA 文案。不要改 URL，不重写正文，不新增页面。

12. 是否有任何 URL 修改建议？
没有可执行 URL 修改建议。当前 `/hoteles/`、`/catalogo/hoteles/`、六个酒店小类页、`/proyectos/hoteles/` 的结构合理。若未来 GSC 证明某个筛选词长期有量，再作为人工确认项讨论，不在本轮执行。

## LATAM localization screening

- Passed: `muebles para hoteles`, `mobiliario hotelero`, `muebles para hotel`, `muebles para habitaciones de hotel`, `muebles para restaurante de hotel`, `muebles para exteriores de hotel`, `muebles para recepción de hotel`, `cotización de muebles para hoteles`.
- Passed with qualification: `sillas para restaurante`, `mesas para restaurante`, `sillas para bar`, `muebles para exteriores`, `muebles para recepción`, `camas para hotel`. Use only under hotel/project/category context.
- Rejected: second-hand, motel/adult, brand-navigation, price/cheap, generic home garden, pure inspiration/Pinterest, and unrelated hotel supplies when furniture intent is weak.

## Safety and limits

- No source code was modified.
- No campaign was created or enabled. No budget, ad group, payment setting or ad creative was created or changed.
- `16` is not Spanish-only because the Google Ads result page disabled language editing and displayed all languages.
- The first forecast-summary export was discarded as evidence because it was campaign-summary data and did not match the four-country historical keyword scope.
