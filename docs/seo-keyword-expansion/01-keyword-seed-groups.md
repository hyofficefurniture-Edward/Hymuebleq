# 扩展关键词种子词组

## 使用规则

- 每组种子词只用于扩展和验证，不代表马上改 Title/H1。
- 高搜索量词不自动成为主词；必须同时满足 B2B 采购意图、页面角色和转化价值。
- 目录页可以像目录，但业务逻辑不能像电商：避免价格、库存、购物车、单件 SKU 语义。
- 项目案例页只做证明，不抢行业主词。
