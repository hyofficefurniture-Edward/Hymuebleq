# Final keyword implementation map

## 使用说明

本文件是下一轮源码修改的主依据。酒店页采用 `18/19`，非酒店页采用 `12/13/14`。所有建议只允许用于 Title、Meta、H1、必要 H2、少量首屏文案、CTA 附近文案、图片 alt 和内链锚文本；不允许改 URL、导航结构、联系方式或案例事实。

字段压缩说明：

- `Title/H1/Meta` 分别给出推荐 Title、推荐 H1、推荐 Meta Description。
- `H2/Hero/Alt` 包含推荐 H2 布局、首屏是否需要微调、图片 alt 建议。
- `Risk/Priority/Exec` 包含页面抢词风险、修改优先级、是否允许下一轮源码执行。
- `Evidence/Fit` 包含 Keyword Planner、SERP、页面角色、LATAM fit 和 Spain-biased risk。
