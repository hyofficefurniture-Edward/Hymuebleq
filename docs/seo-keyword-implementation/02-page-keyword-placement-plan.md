# Page keyword placement plan

## 使用规则

- Title / H1 / 首屏第一段只放 Main keyword 或非常接近的采购表达。
- H2、正文、FAQ、alt、CTA 和内链承接 Secondary、Long-tail、Content-only。
- 高搜索但偏零售的词只能降级，不能进入 Title / H1。
- 目录页使用 reference / catálogo / cotización 语义，不使用 price / stock / cart / SKU。
