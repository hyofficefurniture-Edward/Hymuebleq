# Full-site URL keyword coverage

## 本文件用途

本文件补齐 `docs/url-list.md` 中尚未在 `01-final-keyword-implementation-map.md` 单独规划的现网 URL。

边界：

- 本文件只做关键词规划状态补齐，不修改源码。
- `01-final-keyword-implementation-map.md` 仍是 P0/P1 源码应用主依据。
- 本文件中的 P2 / Observe / No SEO action 页面，不应被下一轮自动纳入源码应用，除非用户明确批准。
- 案例详情页可以保留页面识别词，但不把客户品牌名当作 SEO 主承接词。
- `inherited:` 表示案例详情页继承行业案例页的 proof/reference 语义，不作为独立强主词参与抢词。
